📊 API IBGE – Pipeline de Dados e Relatórios Automatizados

Projeto desenvolvido para coleta, tratamento, análise e geração de relatório de dados do IBGE utilizando linguagem R e API pública.

🛠️ Tecnologias
R httr jsonlite logger yaml here glue RStudio R Markdown
⚙️ Funcionalidades
  • Consumo de dados da API do IBGE
  • Tratamento e padronização das séries temporais
  • Preparação de dados para visualização e métricas estatísticas
  • Execução resiliente com retry automático e Logging estruturado para monitoramento
  • Geração de relatórios
